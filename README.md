## Getting Started

To set up the documentation site locally, follow these steps:

```bash
# clone & checkout
git clone git@github.com:verzaio/verza-docs.git
cd verza-docs
git checkout dev

# install
npm install

# run
npm run dev
```

Open [http://localhost:8081](http://localhost:8081) with your browser.
