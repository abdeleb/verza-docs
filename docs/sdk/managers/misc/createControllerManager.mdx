Creates a controller manager.

### Syntax

```js
createControllerManager(props);
```

### Parameters

| Name    | Type                           | Description       |
| ------- | ------------------------------ | ----------------- |
| `props` | `{ [name: string]: unknown; }` | Controller props. |

### Returns

- [`ControllerManager`](/sdk/types/ControllerManager) - The controller manager.

### Example

```js
// create a controller manager
const manager = createControllerManager({
  status: true,
});

// listen for status changes
manager.events.on('status', status => {
  console.log('new status', status);
});

// change the status
manager.status = false;
```
