Remove a sound from the engine.

### Syntax

```js
engine.audio.removeSound(soundName);
```

### Parameters

| Name        | Type     | Description                     |
| ----------- | -------- | ------------------------------- |
| `soundName` | `string` | The name of the sound to remove |

### Example

```js
engine.audio.removeSound('fire');
```
