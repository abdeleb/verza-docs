Play a sound and automatically dispose of it when it's done playing.

### Syntax

```js
engine.audio.playSound(soundName, options, withId);
```

### Parameters

| Name                   | Type                                      | Description                       |
| ---------------------- | ----------------------------------------- | --------------------------------- |
| `soundName`            | `string`                                  | The name of the sound to create.  |
| `options` _(optional)_ | [`SoundOptions`](/sdk/types/SoundOptions) | The options to use for the sound. |
| `withId` _(optional)_  | `string`                                  | The ID of the sound to create.    |

### Returns

- [`SoundManager`](/sdk/types/SoundManager) - The sound manager.

### Example

```js
const audio = engine.audio.playSound('fire', {
  volume: 0.5,
});

audio.play();
```
