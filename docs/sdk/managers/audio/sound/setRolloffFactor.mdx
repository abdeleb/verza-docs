Set the sound rolloff factor.

### Syntax

```js
sound.setRolloffFactor(loop);
```

### Parameters

| Name            | Type     | Description         |
| --------------- | -------- | ------------------- |
| `rolloffFactor` | `number` | The rolloff factor. |

### Example

```js
sound.setRolloffFactor(1);
```

**Available on:** <ClientTag />
