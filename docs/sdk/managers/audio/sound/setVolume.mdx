Set the sound volume.

### Syntax

```js
sound.setVolume(volume);
```

### Parameters

| Name     | Type     | Description                                     |
| -------- | -------- | ----------------------------------------------- |
| `volume` | `number` | The volume to set. Must be between `0` and `1`. |

### Example

```js
// Set the volume to 50%.
sound.setVolume(0.5);
```
