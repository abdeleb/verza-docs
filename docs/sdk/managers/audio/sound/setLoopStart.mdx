Set the sound loop start.

### Syntax

```js
sound.setLoopStart(loopStart);
```

### Parameters

| Name        | Type     | Description                     |
| ----------- | -------- | ------------------------------- |
| `loopStart` | `number` | The loop start time in seconds. |

### Example

```js
// Set the loop start to 0.5 seconds.
sound.setLoopStart(0.5);
```
