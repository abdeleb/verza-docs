Set the sound maximum distance to fade out.

### Syntax

```js
sound.setMaxDistance(maxDistance);
```

### Parameters

| Name          | Type     | Description                 |
| ------------- | -------- | --------------------------- |
| `maxDistance` | `number` | The sound maximum distance. |

### Example

```js
sound.setMaxDistance(10);
```

**Available on:** <ClientTag />
