Destroy the sound.

:::note Attached to an entity
If the sound is attached to an entity (e.g: Object), it will be destroyed when the entity is destroyed. No need to call `sound.destroy()`.
:::

### Syntax

```js
sound.destroy();
```

### Example

```js
sound.destroy();
```

**Available on:** <ClientTag />
