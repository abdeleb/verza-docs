Set the sound options.

### Syntax

```js
sound.setOptions(loop);
```

### Parameters

| Name      | Type                                      | Description        |
| --------- | ----------------------------------------- | ------------------ |
| `options` | [`SoundOptions`](/sdk/types/SoundOptions) | The sound options. |

### Example

```js
sound.setOptions({
  volume: 0.5,
});
```
