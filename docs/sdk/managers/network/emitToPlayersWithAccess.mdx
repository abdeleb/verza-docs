Emit an event to players with specific access on the server.

### Syntax

```js
engine.network.emitToPlayersWithAccess(event, commandOrAction, data);
```

### Parameters

| Name              | Type                                              | Description                 |
| ----------------- | ------------------------------------------------- | --------------------------- |
| `event`           | `string`                                          | The event name to emit.     |
| `commandOrAction` | `string`                                          | The command or action name. |
| `data`            | [`NetworkEventData`](/sdk/types/NetworkEventData) | The event data. (optional)  |

### Example

```js
engine.network.emitToPlayersWithAccess('onOpenDoor', '+editor', {
  doorId: 1,
  status: true,
});
```

**Available on:** <ClientTag /> <ServerTag />
