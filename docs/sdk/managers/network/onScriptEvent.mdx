Register a listener function to be called when a script event is emitted.

### Syntax

```js
engine.network.onScriptEvent(event, listener);
```

### Parameters

| Name       | Type                                | Description                  |
| ---------- | ----------------------------------- | ---------------------------- |
| `event`    | `string`                            | The event name to subscribe. |
| `listener` | `(data?: NetworkEventData) => void` | The listener.                |

### Returns

- `listener` - The listener.

### Example

```js
engine.network.onScriptEvent('onOpenDoor', (({status})) => {
  if (status) {
    door.open()
  } else {
    door.close()
  }
});
```
