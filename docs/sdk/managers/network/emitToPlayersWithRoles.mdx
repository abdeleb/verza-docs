Emit an event to players with specific roles on the server.

### Syntax

```js
engine.network.emitToPlayersWithRoles(event, roles, data);
```

### Parameters

| Name                | Type                                              | Description             |
| ------------------- | ------------------------------------------------- | ----------------------- |
| `event`             | `string`                                          | The event name to emit. |
| `roles`             | `string[]`                                        | The roles to emit to.   |
| `data` _(optional)_ | [`NetworkEventData`](/sdk/types/NetworkEventData) | The event data.         |

### Example

```js
engine.network.emitToPlayersWithRoles(
  'onOpenDoor',
  ['role_2jd1TjKpCYjhFKDhw8hmTH'],
  {
    doorId: 1,
    status: true,
  },
);
```
