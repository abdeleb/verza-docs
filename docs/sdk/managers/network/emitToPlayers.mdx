Emit an event to all players on the server.

### Syntax

```js
engine.network.emitToPlayers(event, data);
```

### Parameters

| Name    | Type                                              | Description                |
| ------- | ------------------------------------------------- | -------------------------- |
| `event` | `string`                                          | The event name to emit.    |
| `data`  | [`NetworkEventData`](/sdk/types/NetworkEventData) | The event data. (optional) |

### Example

```js
engine.network.emitToPlayers('onOpenDoor', {
  doorId: 1,
  status: true,
});
```

**Available on:** <ClientTag /> <ServerTag />
