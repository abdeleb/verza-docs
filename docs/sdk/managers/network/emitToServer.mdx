Emit an event to the server.

:::note
This method can only be called on the client.
:::

### Syntax

```js
engine.network.emitToServer(event, data);
```

### Parameters

| Name                | Type                                              | Description             |
| ------------------- | ------------------------------------------------- | ----------------------- |
| `event`             | `string`                                          | The event name to emit. |
| `data` _(optional)_ | [`NetworkEventData`](/sdk/types/NetworkEventData) | The event data.         |

### Example

```js
engine.network.emitToServer('onOpenDoor', {
  doorId: 1,
  status: true,
});
```
