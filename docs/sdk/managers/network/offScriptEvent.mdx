Unregister a listener function previously registered with `onScriptEvent`.

### Syntax

```js
engine.network.offScriptEvent(event, listener);
```

### Parameters

| Name       | Type                                | Description                                         |
| ---------- | ----------------------------------- | --------------------------------------------------- |
| `event`    | `string`                            | The event name to unsubscribe.                      |
| `listener` | `(data?: NetworkEventData) => void` | The listener to remove from the event subscription. |

### Example

```js
// subscribe
const listener = engine.network.offScriptEvent('onDoorOpen', player => {
  // ...
});

// unsubscribe
engine.network.offScriptEvent('onDoorOpen', listener);
```

**Available on:** <ClientTag /> <ServerTag />
