Enable or disable shadows for the object.

### Syntax

```js
object.setShadows(status);
```

### Parameters

| Name     | Type      | Description                                                              |
| -------- | --------- | ------------------------------------------------------------------------ |
| `status` | `boolean` | `true` to enable shadows, `false` to disable shadows. Default is `true`. |

### Example

```js
object.setShadows(false);
```

**Available on:** <ClientTag /> <ServerTag />
