Attaches any entity to the object.

### Syntax

```js
object.attach(entity, options);
```

### Parameters

| Name                   | Type                                                    | Description       |
| ---------------------- | ------------------------------------------------------- | ----------------- |
| `entity`               | [`EntityManager`](/sdk/types/EntityManager)             | Entity to attach. |
| `options` _(optional)_ | [`EntityAttachOptions`](/sdk/types/EntityAttachOptions) | Attach options.   |

### Example

```js
const box1 = engine.objects.create('box');
const box2 = engine.objects.create('box', {position: [0, 1, 0]});

box1.attach(box2);
```
