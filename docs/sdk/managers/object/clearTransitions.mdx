Clears all active transitions for the object.

### Syntax

```js
object.clearTransitions();
```

### Example

```js
object.startTransition({
  to: [0, 5, 15],
  durtion: 500,
});

// ...

object.clearTransitions();
```

**Available on:** <ClientTag /> <ServerTag />
