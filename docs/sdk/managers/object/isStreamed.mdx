Checks if an object is currently being streamed.

### Syntax

```js
object.isStreamed();
```

### Returns

- `Promise<boolean>` - Whether the object is currently being streamed.

### Example

```js
const isStreamed = await object.isStreamed();

console.log('is streamed?', isStreamed);
```
