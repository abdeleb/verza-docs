Enable or disable bloom for the object.

### Syntax

```js
object.setBloom(status);
```

### Parameters

| Name     | Type      | Description                                                           |
| -------- | --------- | --------------------------------------------------------------------- |
| `status` | `boolean` | `true` to enable bloom, `false` to disable bloom. Default is `false`. |

### Example

```js
object.setBloom(false);
```

**Available on:** <ClientTag /> <ServerTag />
