Compute the bounding box of the object.

### Syntax

```js
object.computeBoundingBox();
```

### Returns

- [`Promise<Box3>`](/sdk/types/Box3) - A promise that resolves with the bounding box of the object.

### Example

```js
const boundingBox = await object.computeBoundingBox();
```
