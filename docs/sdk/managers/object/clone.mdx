Creates a new object by cloning an existing object with the same properties.

### Syntax

```js
object.clone(withId);
```

### Parameters

| Name     | Type     | Description                          |
| -------- | -------- | ------------------------------------ |
| `withId` | `string` | The ID of the new object (optional). |

### Returns

- `ObjectManager` - The created object manager.

### Example

```js
const clonedObject = object.clone('new-id');
```

**Available on:** <ClientTag /> <ServerTag />
