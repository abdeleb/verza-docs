Set the position of the object.

### Syntax

```js
object.setPosition(position);
```

### Parameters

| Name       | Type                                                                         | Description      |
| ---------- | ---------------------------------------------------------------------------- | ---------------- |
| `position` | [`Vector3`](/sdk/types/Vector3) or [`Vector3Array`](/sdk/types/Vector3Array) | The new position |

### Example

```js
object.setPosition([0, 5, 0]);
```

**Available on:** <ClientTag /> <ServerTag />
