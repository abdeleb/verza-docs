Set the collision type of the object.

### Syntax

```js
object.setCollision(type);
```

### Parameters

| Name   | Type                                                    | Description            |
| ------ | ------------------------------------------------------- | ---------------------- |
| `type` | [`EntityCollisionType`](/sdk/types/EntityCollisionType) | The new collision type |

### Example

```js
object.setCollision('dynamic');
```

**Available on:** <ClientTag /> <ServerTag />
