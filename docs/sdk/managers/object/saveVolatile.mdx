Same as [`object.save`](./save) but with a delay.

### Syntax

```js
object.saveVolatile();
```

### Returns

- `Promise<void>`: A promise that resolves when the object is saved.

### Example

```js
await object.saveVolatile();
```

**Available on:** <ClientTag /> <ServerTag />
