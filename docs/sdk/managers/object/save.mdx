Saves an object to the Verza servers, making it permanent.

This method is used to store objects long-term.

### Syntax

```js
object.save();
```

### Returns

- `Promise<void>`: A promise that resolves when the object is saved.

### Example

```js
await object.save();
```

**Available on:** <ClientTag /> <ServerTag />
