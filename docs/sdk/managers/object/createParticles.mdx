Creates a particle effect attached to the object.

### Syntax

```js
object.createParticles(options, withId);
```

### Parameters

| Name                   | Type                                            | Description                               |
| ---------------------- | ----------------------------------------------- | ----------------------------------------- |
| `options` _(optional)_ | [`ParticleOptions`](/sdk/types/ParticleOptions) | The options for the particles effect.     |
| `withId` _(optional)_  | `string`                                        | The ID of the particles effect to create. |

### Returns

- [`ParticlesManager`](/sdk/types/ParticlesManager) - The particles manager.

### Example

```js
const particles = object.createParticles({
  position: [0, 2, 0],
  amount: 50,
});
```
