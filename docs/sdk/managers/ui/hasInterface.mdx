Checks whether the interface has been added to UI.

### Syntax

```js
engine.ui.hasInterface(interface);
```

### Parameters

| Name        | Type     | Description        |
| ----------- | -------- | ------------------ |
| `interface` | `string` | The interface name |

### Returns

- `boolean` - Whether the interface has been added to UI.

### Example

```js
engine.ui.hasInterface('my-interface');
```
