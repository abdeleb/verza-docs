Removes an interface from the UI.

### Syntax

```js
engine.ui.removeInterface(interface);
```

### Parameters

| Name        | Type     | Description        |
| ----------- | -------- | ------------------ |
| `interface` | `string` | The interface name |

### Example

```js
engine.ui.removeInterface('my-interface');
```

**Available on:** <ClientTag />
