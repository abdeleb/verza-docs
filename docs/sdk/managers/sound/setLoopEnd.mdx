Set the sound loop end.

### Syntax

```js
sound.setLoopEnd(loopEnd);
```

### Parameters

| Name      | Type     | Description              |
| --------- | -------- | ------------------------ |
| `loopEnd` | `number` | The loop end in seconds. |

### Example

```js
// Set the loop end to 2.5 seconds.
sound.setLoopEnd(2.5);
```
