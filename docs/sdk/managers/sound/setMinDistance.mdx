Set the sound minimum distance to start to fade out.

### Syntax

```js
sound.setMinDistance(minDistance);
```

### Parameters

| Name          | Type     | Description                        |
| ------------- | -------- | ---------------------------------- |
| `minDistance` | `number` | The minimum distance of the sound. |

### Example

```js
sound.setMinDistance(2);
```

**Available on:** <ClientTag />
