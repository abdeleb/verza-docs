Set the sound playback rate.

### Syntax

```js
sound.setPlaybackRate(playbackRate);
```

### Parameters

| Name           | Type     | Description               |
| -------------- | -------- | ------------------------- |
| `playbackRate` | `number` | The playback rate to set. |

### Example

```js
// Set the playback rate to x2.
sound.setPlaybackRate(2);
```
