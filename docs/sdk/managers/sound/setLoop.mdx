Set the sound loop.

### Syntax

```js
sound.setLoop(loop);
```

### Parameters

| Name   | Type                                | Description      |
| ------ | ----------------------------------- | ---------------- |
| `loop` | [`SoundLoop`](/sdk/types/SoundLoop) | The loop to set. |

### Example

```js
sound.setLoop('repeat');

// or

sound.setLoop('once');
```
