Sets the fog color.

### Syntax

```js
engine.world.setFogColor(color);
```

### Parameters

| Name    | Type                                | Description                    |
| ------- | ----------------------------------- | ------------------------------ |
| `color` | [`ColorType`](/sdk/types/ColorType) | The color of the global light. |

### Example

```js
engine.world.setFogStatus(true);
engine.world.setFogColor('red');
```
