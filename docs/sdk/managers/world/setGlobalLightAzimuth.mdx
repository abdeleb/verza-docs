Sets the azimuth of the global light.

### Syntax

```js
engine.world.setGlobalLightAzimuth(intensity);
```

### Parameters

| Name      | Type     | Description                                         |
| --------- | -------- | --------------------------------------------------- |
| `azimuth` | `number` | The azimuth of the global light in degrees (0-360). |

### Example

```js
engine.world.setGlobalLightAzimuth(90);
```
