Creates a proximity action attached to an object.

### Syntax

```js
engine.world.createObjectProximityAction(object, action, withId);
```

### Parameters

| Name                  | Type                                                          | Description                                   |
| --------------------- | ------------------------------------------------------------- | --------------------------------------------- |
| `object`              | `string` or [`ObjectManager`](/sdk/types/ObjectManager)       | The object to attach the proximity action to. |
| `action` _(optional)_ | [`ProximityActionOptions`](/sdk/types/ProximityActionOptions) | The proximity action options.                 |
| `withId` _(optional)_ | `string`                                                      | The id of the proximity action.               |

### Returns

- [`ProximityActionManager`](/sdk/types/ProximityActionManager) - The proximity action manager.

### Example

```js
// create a proximity action
engine.world.createObjectProximityAction(
  player,
  {
    position: [3, 2, 3],
    label: 'Nice music',
    description: 'Play a sound',
    key: 'KeyF',
    distance: 3,
  },
  'my-custom-id',
);

// listen for proximity action triggers
engine.world.on('onProximityActionTriggered', event => {
  console.log('Proximity action triggered', event);
});
```
