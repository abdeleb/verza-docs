Deletes a proximity action.

### Syntax

```js
engine.world.deleteProximityAction(actionId);
```

### Parameters

| Name       | Type     | Description                     |
| ---------- | -------- | ------------------------------- |
| `actionId` | `string` | The id of the action to delete. |

### Example

```js
engine.world.deleteProximityAction('action-id');
```
