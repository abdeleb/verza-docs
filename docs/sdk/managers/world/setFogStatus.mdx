Sets the fog status.

### Syntax

```js
engine.world.setFogStatus(status);
```

### Parameters

| Name     | Type      | Description                                   |
| -------- | --------- | --------------------------------------------- |
| `status` | `boolean` | `true` to enable fog, `false` to disable fog. |

### Example

```js
engine.world.setFogStatus(true);
```
