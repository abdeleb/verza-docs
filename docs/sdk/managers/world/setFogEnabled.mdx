Sets the fog status.

### Syntax

```js
engine.world.setFogEnabled(enabled);
```

### Parameters

| Name      | Type      | Description                                   |
| --------- | --------- | --------------------------------------------- |
| `enabled` | `boolean` | `true` to enable fog, `false` to disable fog. |

### Example

```js
engine.world.setFogEnabled(true);
```
