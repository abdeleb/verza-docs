Sets the fog density.

### Syntax

```js
engine.world.setFogDensity(color);
```

### Parameters

| Name      | Type     | Description      |
| --------- | -------- | ---------------- |
| `density` | `number` | The fog density. |

### Example

```js
engine.world.setFogDensity(0.005);
```
