Creates a new object with the specified properties.

### Syntax

```js
engine.objects.create(type, props);
```

### Parameters

| Name    | Type                                                | Description      |
| ------- | --------------------------------------------------- | ---------------- |
| `type`  | [`ObjectType`](/sdk/types/ObjectType)               | The object type  |
| `props` | [`CreateObjectProps`](/sdk/types/CreateObjectProps) | The object props |

### Returns

- [`ObjectManager`](/sdk/types/ObjectManager) - The created object manager.

### Example

```js
const box = engine.objects.create('box', {
  color: 'red',
  width: 1,
  height: 1,
  depth: 1,
  position: [1, 2, 3],
});
```
