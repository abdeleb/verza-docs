Enables or disables the display of translation, rotation and scaling axes while editing objects.

### Syntax

```js
engine.objects.setEditAxes(axes);
```

### Parameters

| Name   | Type                                          | Description                                                                                 |
| ------ | --------------------------------------------- | ------------------------------------------------------------------------------------------- |
| `axes` | [`ObjectEditAxes`](/sdk/types/ObjectEditAxes) | The object's editing axes, used to manipulate its position and rotation while in edit mode. |

### Example

```js
// show only translation axes
engine.objects.setEditAxes({
  showX: true,
  showY: true,
  showZ: true,

  showRX: false,
  showRY: false,
  showRZ: false,

  showSX: false,
  showSY: false,
  showSZ: false,
});
```
