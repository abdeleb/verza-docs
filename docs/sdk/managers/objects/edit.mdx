Begins editing the specified object.

### Syntax

```js
objects.edit(object);
```

### Parameters

| Name     | Type            | Description         |
| -------- | --------------- | ------------------- |
| `object` | `ObjectManager` | The object to edit. |

### Example

```js
const box = engine.objects.create('box');

engine.objects.edit(box);
```

**Available on:** <ClientTag />
