Sets the nametags distance for all players.

:::caution PERSISTENT METHOD
This method is persistent. It will not reset when a server restarts.
:::

### Syntax

```js
engine.players.setNametagsDistance(distance);
```

### Parameters

| Name       | Type     | Description                                                       |
| ---------- | -------- | ----------------------------------------------------------------- |
| `distance` | `number` | The distance where nametags are visible. Maximum value is `1024`. |

### Returns

- `Promise<void>` - A promise that resolves when the nametags distance has been set.

### Example

```js
engine.players.setNametagsDistance(128);
```

**Available on:** <ClientTag /> <ServerTag />
