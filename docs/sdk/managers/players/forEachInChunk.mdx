Iterates for each player in specified chunk.

### Syntax

```js
engine.players.forEachInChunk(callback);
```

### Parameters

| Name       | Type                                 | Description |
| ---------- | ------------------------------------ | ----------- |
| `callback` | `(entity: PlayerManager) => boolean` | Callback    |

### Example

```js
engine.players.forEachInChunk(`0_0_0_0`, player => {
  player.sendMessage('Hello world!');
});
```

**Available on:** <ClientTag /> <ServerTag />
