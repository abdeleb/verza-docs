Creates a detached player entity.

### Syntax

```js
engine.players.create(playerId);
```

### Parameters

| Name       | Type     | Description   |
| ---------- | -------- | ------------- |
| `playerId` | `number` | The player ID |

### Returns

- [`PlayerManager`](/sdk/types/PlayerManager) - The player manager

### Example

```js
const player = engine.players.create(1212);

const banStatus = await player.getBanStatus();

console.log('Is banned:', banStatus.banned);
```
