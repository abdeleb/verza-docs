Gets a value from the store.

### Syntax

```js
store.get(key);
```

### Parameters

| Name  | Type     | Description                  |
| ----- | -------- | ---------------------------- |
| `key` | `string` | The key of the value to get. |

### Returns

- `Promise<any>` - A promise that resolves with the value.

### Example

```js
const score = await store.get('player1');

console.log('Player 1 score:', score);
```
