Deletes a value from the store.

### Syntax

```js
store.delete(key);
```

### Parameters

| Name  | Type     | Description                     |
| ----- | -------- | ------------------------------- |
| `key` | `string` | The key of the value to delete. |

### Returns

- `Promise<void>` - A promise that resolves when the value is deleted.

### Example

```js
await store.delete('player1');
```
