Stop the particle effect.

### Syntax

```js
particles.stop(instant);
```

### Parameters

| Name                   | Type      | Description                                                          |
| ---------------------- | --------- | -------------------------------------------------------------------- |
| `instant` _(optional)_ | `boolean` | If `true`, the particle effect will stop instantly. Default: `false` |

### Example

```js
particles.stop();
```

**Available on:** <ClientTag />
