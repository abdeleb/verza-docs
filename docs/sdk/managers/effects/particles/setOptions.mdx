Set the particle effect options.

### Syntax

```js
particles.setOptions(options, respawn);
```

### Parameters

| Name                   | Type                                              | Description                                         |
| ---------------------- | ------------------------------------------------- | --------------------------------------------------- |
| `options`              | [`ParticlesOptions`](/sdk/types/ParticlesOptions) | The options for the particles effect.               |
| `respawn` _(optional)_ | `boolean`                                         | Whether to respawn the particles. Default: `false`. |

### Example

```js
particles.setOptions(
  {
    color: 'red',
  },
  true,
);
```
