Play the particle effect.

### Syntax

```js
particles.play(reset, options, respawn);
```

### Parameters

| Name                   | Type                                              | Description                                               |
| ---------------------- | ------------------------------------------------- | --------------------------------------------------------- |
| `reset` _(optional)_   | `boolean`                                         | Whether to stop the particles instantly. Default: `false` |
| `options` _(optional)_ | [`ParticlesOptions`](/sdk/types/ParticlesOptions) | The options for the particles effect.                     |
| `respawn` _(optional)_ | `boolean`                                         | Whether to respawn the particles. Default: `false`        |

### Example

```js
particles.play(true);
```

**Available on:** <ClientTag />
