Destroy the particle effect.

:::note Attached to an entity
If the particles effect is attached to an entity (e.g: Object), it will be destroyed when the entity is destroyed. No need to call `particles.destroy()`.
:::

### Syntax

```js
particles.destroy();
```

### Example

```js
particles.destroy();
```

**Available on:** <ClientTag />
