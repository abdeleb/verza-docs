Play a particle effect at a specific position.

### Syntax

```js
engine.effects.playParticlesAtPosition(position, options, withId);
```

### Parameters

| Name       | Type                                                                         | Description                                          |
| ---------- | ---------------------------------------------------------------------------- | ---------------------------------------------------- |
| `position` | [`Vector3Array`](/sdk/types/Vector3Array) or [`Vector3`](/sdk/types/Vector3) | The position to play the particles at.               |
| `options`  | [`ParticleOptions`](/sdk/types/ParticleOptions)                              | The options for the particles to create.             |
| `withId`   | `string`                                                                     | The ID of the particles effect to create. (optional) |

### Returns

- [`ParticlesManager`](/sdk/types/ParticlesManager) - The particles manager.

### Example

```js
const particles = engine.audio.playParticlesAtLocation([0, 2, 0], {
  amount: 50,
});
```
