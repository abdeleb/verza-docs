Create a particle effect attached to an object.

### Syntax

```js
engine.effects.createObjectParticles(object, options, withId);
```

### Parameters

| Name                   | Type                                                    | Description                                   |
| ---------------------- | ------------------------------------------------------- | --------------------------------------------- |
| `object`               | `string` or [`ObjectManager`](/sdk/types/ObjectManager) | The object to attach the particles effect to. |
| `options` _(optional)_ | [`ParticlesOptions`](/sdk/types/ParticlesOptions)       | The options for the particles effect.         |
| `withId` _(optional)_  | `string`                                                | The ID of the particles effect to create.     |

### Returns

- [`ParticlesManager`](/sdk/types/ParticlesManager) - The particles manager.

### Example

```js
const particles = engine.effects.createObjectParticles(object, {
  position: [0, 2, 0],
  amount: 50,
});
```
