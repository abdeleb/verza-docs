Unregisters a command or action.

### Syntax

```js
engine.commands.unregister(commandOrAction);
```

### Parameters

| Name              | Type                            | Description                         |
| ----------------- | ------------------------------- | ----------------------------------- |
| `commandOrAction` | [`Command`](/sdk/types/Command) | The command or action to unregister |

### Example

```js
engine.commands.unregister(commandOrAction);
```

**Available on:** <ClientTag /> <ServerTag />
