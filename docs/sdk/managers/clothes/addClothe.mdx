Adds a clothe item to the engine.

:::caution Heads Up!
This method do not add the clothes to the player, it only adds it to the clothes system. To add the clothes to the player, use [`player.addClothes`](/sdk/managers/player/addClothes).
:::

### Syntax

```js
engine.clothes.addClothe(item);
```

### Parameters

| Name   | Type                                  | Description                           |
| ------ | ------------------------------------- | ------------------------------------- |
| `item` | [`ClotheItem`](/sdk/types/ClotheItem) | The clothe item to add to the engine. |

### Example

```js
engine.clothes.addClothe({
  id: 'beach-tshirt',
  url: 'https://example.com/beach-tshirt.glb',
  name: 'Beach T-Shirt',
  masks: ['top'],
});
```
