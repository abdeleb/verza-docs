Removes a skin mask from the engine

### Syntax

```js
engine.clothes.removeSkinMask(maskId);
```

### Parameters

| Name     | Type     | Description                        |
| -------- | -------- | ---------------------------------- |
| `maskId` | `string` | The ID of the skin mask to remove. |

### Example

```js
engine.clothes.removeSkinMask('top');
```
