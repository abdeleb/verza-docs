Sets the global voicechat mode.

:::caution PERSISTENT METHOD
This method is persistent. It will not reset when a server restarts.
:::

### Syntax

```js
engine.voicechat.setMode(mode);
```

### Parameters

| Name   | Type                                              | Description                |
| ------ | ------------------------------------------------- | -------------------------- |
| `mode` | [VoicechatModeType](/sdk/types/VoicechatModeType) | The voicechat mode to set. |

### Example

```js
engine.voicechat.setMode(false);
```

**Available on:** <ClientTag /> <ServerTag />
