Set whether or not the player is visible.

### Syntax

```js
player.setVisible(visible);
```

### Parameters

| Name      | Type      | Description                                                            |
| --------- | --------- | ---------------------------------------------------------------------- |
| `visible` | `boolean` | Determines whether the player is visible (`true`) or hidden (`false`). |

### Example

```js
player.setVisible(false);
```

**Available on:** <ClientTag /> <ServerTag />
