Attaches any entity to the player.

### Syntax

```js
player.attach(entity, options);
```

### Parameters

| Name                   | Type                                                    | Description       |
| ---------------------- | ------------------------------------------------------- | ----------------- |
| `entity`               | [`EntityManager`](/sdk/types/EntityManager)             | Entity to attach. |
| `options` _(optional)_ | [`EntityAttachOptions`](/sdk/types/EntityAttachOptions) | Attach options.   |

### Example

```js
const box = engine.objects.create('box');

player.attach(box);
```

**Available on:** <ClientTag /> <ServerTag />
