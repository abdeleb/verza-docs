Gets the player's clothes.

### Syntax

```js
player.getClothes();
```

### Returns

- `Promise<PlayerClotheItem[]>` - Returns a promise containing an array of [PlayerClotheItem](/sdk/types/PlayerClotheItem) objects.

### Example

```js
const clothes = await player.getClothes();
```
