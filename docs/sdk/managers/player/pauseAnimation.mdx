Pauses an animation on the player.

### Syntax

```js
player.pauseAnimation(animId);
```

### Parameters

| Name     | Type     | Description                       |
| -------- | -------- | --------------------------------- |
| `animId` | `string` | The ID of the animation to pause. |

### Example

```js
player.pauseAnimation('dance');
```

**Available on:** <ClientTag /> <ServerTag />
