Sets the player's chat enabled status.

### Syntax

```js
player.setChatEnabled(enabled);
```

### Parameters

| Name      | Type      | Description                                         |
| --------- | --------- | --------------------------------------------------- |
| `enabled` | `boolean` | Whether the player's chat should be enabled or not. |

### Example

```js
player.setChatEnabled(false);
```

**Available on:** <ServerTag />
