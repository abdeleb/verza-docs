Resumes an animation on the player.

### Syntax

```js
player.resumeAnimation(animId);
```

### Parameters

| Name     | Type     | Description                       |
| -------- | -------- | --------------------------------- |
| `animId` | `string` | The ID of the animation to resume |

### Example

```js
player.resumeAnimation('dance');
```

**Available on:** <ClientTag /> <ServerTag />
