Adds a role to the player.

### Syntax

```js
player.addRole(role);
```

### Returns

- `Promise<void>` - A promise that resolves when the role has been added.

### Parameters

| Name   | Type     | Description                    |
| ------ | -------- | ------------------------------ |
| `role` | `string` | The role to add to the player. |

### Example

```js
await player.addRole('role_2jd1TjKpCYjhFKDhw8hmTH');
```
