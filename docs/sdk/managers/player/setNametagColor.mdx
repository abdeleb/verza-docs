Sets the player's nametag color.

:::note
This method is available on the <ServerTag /> but is not synchronized to all players.

You can easily do so by using network events.
:::

### Syntax

```js
player.setNametagColor(name);
```

### Parameters

| Name   | Type                                | Description            |
| ------ | ----------------------------------- | ---------------------- |
| `name` | [`ColorType`](/sdk/types/ColorType) | The new nametag color. |

### Example

```js
player.setNametagColor('red');
```

**Available on:** <ClientTag /> <ServerTag />
