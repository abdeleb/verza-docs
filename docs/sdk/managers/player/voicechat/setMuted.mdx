Sets the player's muted status.

### Syntax

```js
player.setMuted(muted);
```

### Parameters

| Name    | Type      | Description                                |
| ------- | --------- | ------------------------------------------ |
| `muted` | `boolean` | Whether the player should be muted or not. |

### Example

```js
player.setMuted(true);
```

**Available on:** <ServerTag />
