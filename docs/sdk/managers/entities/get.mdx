Retrieves the entity with the specified ID. Returns `null` if the entity does not exist.

### Syntax

```js
entities.get(id);
```

### Parameters

| Name | Type                 | Description |
| ---- | -------------------- | ----------- |
| `id` | `number` or `string` | Entity ID   |

### Returns

- [`EntityManager`](/sdk/types/EntityManager) - The entity with the specified ID or `null` if the entity does not exist.

### Example

```js
console.log('player', engine.players.get(1));
```

**Available on:** <ClientTag /> <ServerTag />
