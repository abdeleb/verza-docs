Creates a new array with the results of calling a provided function on every entity, similar to [`Array.map`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/map). |

### Syntax

```js
entities.map(callback);
```

### Parameters

| Name       | Type                                | Description |
| ---------- | ----------------------------------- | ----------- |
| `callback` | `<T>(entity: EntityManager) => T[]` | Callback    |

### Example

```js
const names = engine.players.map(player => player.name);

console.log('players name', names);
```
