Checks whether an entity with the specified ID exists.

### Syntax

```js
entities.is(id);
```

### Parameters

| Name | Type                 | Description |
| ---- | -------------------- | ----------- |
| `id` | `number` or `string` | Entity ID   |

### Example

```js
console.log(
  'player with id 1 is ' + (engine.players.is(1) ? 'online' : 'offline'),
);
```
