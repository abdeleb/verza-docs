Returns the first entity that satisfies the provided testing function or `null`, similar to [`Array.find`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/find). If no entity matches the test, the method returns `null`.

### Syntax

```js
entities.filter(callback);
```

### Parameters

| Name       | Type                                       | Description |
| ---------- | ------------------------------------------ | ----------- |
| `callback` | `(entity: EntityManager) => EntityManager` | Callback    |

### Example

```js
const VIP_ROLE = 'role_2jd1TjKpCYjhFKDhw8hmTH';

const vipPlayer = engine.players.find(player => player.name.includes(VIP_ROLE));

console.log('vip player', vipPlayer);
```
