Adds animations to the engine.

:::caution Heads Up!
This method do not play animations to the player, it only adds it to the engine. To play animations to the player, use [`player.playAnimation`](/sdk/managers/player/playAnimation).
:::

:::note Format
The animations file must be in the [glTF](https://www.khronos.org/gltf/) format, and animations must be exported as NLA Actions (NLA Tracks).

You can use [Blender](https://www.blender.org/) to create animations and export them to the glTF format.
:::

### Syntax

```js
engine.animations.addAnimations(url);
```

### Parameters

| Name  | Type     | Description                           |
| ----- | -------- | ------------------------------------- |
| `url` | `string` | The URL of the animation file to add. |

### Returns

- [`Promise<AnimationInfo[]>`](/sdk/types/AnimationInfo) - A promise that resolves to an array of `AnimationInfo` containing information about the animations that were added.

### Example

```js
const anims = await engine.animations.addAnimations(
  'https://example.com/animation.glb',
);

console.log('Animations added:', anims);
```
