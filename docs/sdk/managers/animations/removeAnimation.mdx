Removes an animation by id.

### Syntax

```js
engine.animations.removeAnimation(animId);
```

### Parameters

| Name     | Type     | Description                        |
| -------- | -------- | ---------------------------------- |
| `animId` | `string` | The ID of the animation to remove. |

### Example

```js
const anims = await engine.animations.removeAnimation('dance');
```
