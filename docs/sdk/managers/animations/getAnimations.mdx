Gets all animations.

### Syntax

```js
engine.animations.getAnimations();
```

### Returns

- [`Promise<AnimationInfo[]>`](/sdk/types/AnimationInfo) - Returns a promise containing an array of `AnimationInfo` objects.

### Example

```js
const animsInfo = await engine.animations.getAnimations();

console.log('Animations:', animsInfo);
```
