Gets all animations.

### Syntax

```js
engine.animations.getAnimations();
```

### Returns

- `Promise<AnimationInfo[]>` - Returns a promise containing an array of [AnimationInfo](/sdk/types/AnimationInfo) objects.

### Example

```js
const animsInfo = await engine.animations.getAnimations();

console.log('Animations:', animsInfo);
```
