This event is triggered when a player leaves a sensor.

### Parameters

| Name     | Type                                        | Description                         |
| -------- | ------------------------------------------- | ----------------------------------- |
| `player` | [`PlayerManager`](/sdk/types/PlayerManager) | The player that entered the sensor. |

### Example

```js
object.events.on('onLeaveSensor', player => {
  console.log('Player left sensor', player);
});
```

**Available on:** <ClientTag />
