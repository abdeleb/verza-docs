This event is triggered when the pointer moves over an object.

### Parameters

| Name    | Type                                                  | Description              |
| ------- | ----------------------------------------------------- | ------------------------ |
| `event` | [`ObjectPointerEvent`](/sdk/types/ObjectPointerEvent) | The pointer event object |

### Example

```js
object.events.on('onPointerMove', event => {
  console.log('Pointer moved:', event);
});
```
