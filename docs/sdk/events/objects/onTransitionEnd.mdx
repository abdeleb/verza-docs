This event is triggered when a transition related to an object ends.

### Parameters

| Name | Type               | Description       |
| ---- | ------------------ | ----------------- |
| `id` | `number or string` | The transition ID |

### Example

```js
object.events.on('onCameraTransitionEnd', id => {
  console.log('Transition ended:', id);
});
```

**Available on:** <ClientTag />
