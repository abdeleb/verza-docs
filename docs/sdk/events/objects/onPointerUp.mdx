This event is triggered when the pointer is released after being pressed down on an object.

### Parameters

| Name    | Type                 | Description              |
| ------- | -------------------- | ------------------------ |
| `event` | `ObjectPointerEvent` | The pointer event object |

### Example

```js
object.events.on('onPointerUp', event => {
  console.log('Pointer released:', event);
});
```
