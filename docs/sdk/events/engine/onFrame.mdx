This event is triggered every frame.

### Parameters

| Name    | Type     | Description                               |
| ------- | -------- | ----------------------------------------- |
| `delta` | `number` | The time in seconds since the last frame. |

### Example

```js
engine.events.on('onFrame', delta => {
  console.log('Frame delta:', delta);
});
```

**Available on:** <ClientTag /> <ServerTag />
