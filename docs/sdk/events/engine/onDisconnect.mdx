This event is triggered when a script is disconnected from the server.

:::caution
This event is only triggered when the script is disconnected by the server. Use `onDestroy` for any cleanup code.
:::

### Example

```js
engine.events.on('onDisconnect', () => {
  console.log('Script disconnected from server!');
});
```

**Available on:** <ClientTag /> <ServerTag />
