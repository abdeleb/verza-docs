This event is fired when all resources are ready to be used.

:::note HOW TO TRIGGER
To trigger this event you should call [`engine.requestResourcesCheck()`](/sdk/managers/engine/requestResourcesCheck).
:::

### Example

```js
engine.events.on('onResourcesReady', () => {
  console.log;
});
```

**Available on:** <ClientTag />
