This event is triggered when a key event occurs.

### Parameters

| Name    | Type                              | Description   |
| ------- | --------------------------------- | ------------- |
| `event` | [`KeyEvent`](/sdk/types/KeyEvent) | The key event |

### Example

```js
engine.events.on('onKeyEvent', event => {
  console.log('Key event:', event);
});
```
