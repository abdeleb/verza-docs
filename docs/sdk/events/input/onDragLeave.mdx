This event is triggered when a dragged item leaves the canvas.

### Parameters

| Name    | Type                                | Description     |
| ------- | ----------------------------------- | --------------- |
| `event` | [`DragEvent`](/sdk/types/DragEvent) | The drag event. |

### Example

```js
engine.events.on('onDragLeave', () => {
  console.log('Dragged item left the drop target');
});
```
