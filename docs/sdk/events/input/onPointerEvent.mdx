This event is triggered when a pointer event occurs.

### Parameters

| Name    | Type                                      | Description       |
| ------- | ----------------------------------------- | ----------------- |
| `event` | [`PointerEvent`](/sdk/types/PointerEvent) | The pointer event |

### Example

```js
engine.events.on('onPointerEvent', status => {
  console.log('Pointer event:', event);
});
```
