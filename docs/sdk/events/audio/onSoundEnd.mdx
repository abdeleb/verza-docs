This event is triggered when a sound ends.

### Parameters

| Name    | Type                                  | Description                   |
| ------- | ------------------------------------- | ----------------------------- |
| `event` | [`SoundEvent`](/sdk/types/SoundEvent) | The event that was triggered. |

### Example

```js
engine.events.on('onSoundEnd', event => {
  console.log('Sound ended:', event);
});
```
