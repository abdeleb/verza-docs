This event is triggered when a camera transition ends.

### Parameters

| Name | Type     | Description       |
| ---- | -------- | ----------------- |
| `id` | `number` | The transition ID |

### Example

```js
engine.events.on('onCameraTransitionEnd', id => {
  console.log('the camera transition with ID ' + id + ' has ended');
});
```

**Available on:** <ClientTag />
