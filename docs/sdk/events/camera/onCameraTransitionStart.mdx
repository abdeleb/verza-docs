This event is triggered when a camera transition starts.

### Parameters

| Name | Type               | Description       |
| ---- | ------------------ | ----------------- |
| `id` | `number or string` | The transition ID |

### Example

```js
engine.events.on('onCameraTransitionStart', id => {
  console.log('the camera transition with ID ' + id + ' has started');
});
```

**Available on:** <ClientTag />
