This event is triggered when a player moves their head.

### Parameters

| Name    | Type         | Description                  |
| ------- | ------------ | ---------------------------- |
| `euler` | `EulerArray` | The Euler angles of the head |

### Example

```js
engine.localPlayer.events.on('onHeadMove', euler => {
  console.log('Head moved: Euler angles:', euler);
});
```

**Available on:** <ClientTag />
