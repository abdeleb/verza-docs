This event is triggered when a player changes its gender.

### Parameters

| Name     | Type                                            | Description                     |
| -------- | ----------------------------------------------- | ------------------------------- |
| `gender` | [`CharacterGender`](/sdk/types/CharacterGender) | The new gender of the character |

### Example

```js
engine.localPlayer.events.on('onGenderChange', gender => {
  console.log('new gender', gender);
});
```

**Available on:** <ClientTag />
