This event is triggered when a player enters a sensor.

:::note
If you want to detect when a player enters a specific sensor, you should use [`object.events(onEnterSensor)`](/sdk/events/objects/onEnterSensor) instead.
:::

### Parameters

| Name     | Type                                        | Description                          |
| -------- | ------------------------------------------- | ------------------------------------ |
| `object` | [`ObjectManager`](/sdk/types/ObjectManager) | The object that triggered the sensor |

### Example

```js
player.events.on('onEnterSensor', object => {
  console.log('Player entered sensor', object);
});
```
