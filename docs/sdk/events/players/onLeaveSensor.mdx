This event is triggered when a player leaves a sensor.

### Parameters

| Name     | Type                                        | Description                          |
| -------- | ------------------------------------------- | ------------------------------------ |
| `object` | [`ObjectManager`](/sdk/types/ObjectManager) | The object that triggered the sensor |

### Example

```js
player.events.on('onLeaveSensor', object => {
  console.log('Player left sensor', object);
});
```

**Available on:** <ClientTag />
