This event is triggered when a dragged item is dropped on the canvas.

### Parameters

| Name    | Type                                        | Description                |
| ------- | ------------------------------------------- | -------------------------- |
| `event` | [`DragEvent`](/sdk/types/DragEvent)         | The drag event.            |
| `files` | [`FileTransfer[]`](/sdk/types/FileTransfer) | The dropped files, if any. |

### Example

```js
engine.events.on('onDrop', (_, files) => {
  console.log('Dropped item:', files?.length ? files : 'No files provided');
});
```
