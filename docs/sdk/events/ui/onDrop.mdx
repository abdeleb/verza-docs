This event is triggered when a dragged item is dropped on a target.

### Parameters

| Name   | Type                                      | Description                 |
| ------ | ----------------------------------------- | --------------------------- |
| `file` | [`FileTransfer`](/sdk/types/FileTransfer) | The dropped file (optional) |

### Example

```js
engine.events.on('onDrop', file => {
  console.log('Dropped item:', file ? file : 'No file provided');
});
```

**Available on:** <ClientTag />
