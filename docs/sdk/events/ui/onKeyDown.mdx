This event is triggered when a key is pressed down.

### Parameters

| Name    | Type       | Description   |
| ------- | ---------- | ------------- |
| `event` | `KeyEvent` | The key event |

### Example

```js
engine.events.on('onKeyDown', event => {
  console.log('Key pressed down:', event);
});
```
