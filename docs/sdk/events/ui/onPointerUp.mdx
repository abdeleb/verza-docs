This event is triggered when the pointer is released.

### Parameters

| Name    | Type           | Description       |
| ------- | -------------- | ----------------- |
| `event` | `PointerEvent` | The pointer event |

### Example

```js
engine.events.on('onPointerMove', event => {
  console.log('Pointer moved:', event);
});
```
