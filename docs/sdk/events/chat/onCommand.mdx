This event is triggered when a player sends a command.

:::caution
We recommend using the built-in command system instead of relying on this event, unless you are confident in what you're doing.
:::

### Parameters

| Name      | Type     | Description                      |
| --------- | -------- | -------------------------------- |
| `command` | `string` | The command, excluding the slash |

### Example

```js
engine.events.on('onCommand', command => {
  console.log(`command`, command);
});
```

**Available on:** <ClientTag />
