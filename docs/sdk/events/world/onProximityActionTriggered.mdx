This event is triggered when a proximity action is triggered.

### Parameters

| Name    | Type                                                      | Description                |
| ------- | --------------------------------------------------------- | -------------------------- |
| `event` | [`ProximityActionEvent`](/sdk/types/ProximityActionEvent) | The proximity action event |

### Example

```js
engine.events.on('onProximityActionTriggered', event => {
  console.log('Proximity action triggered', event);
});
```
