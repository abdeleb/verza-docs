This event is triggered when an entity is selected.

### Parameters

| Name         | Type               | Description                          |
| ------------ | ------------------ | ------------------------------------ |
| `intersects` | `IntersectsResult` | The intersection result of selection |

### Example

```js
engine.entityManager.events.on('onEntitySelected', intersects => {
  console.log('Entity selected:', intersects);
});
```

**Available on:** <ClientTag />
