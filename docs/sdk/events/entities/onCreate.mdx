This event is triggered when an entity is created.

### Parameters

| Name     | Type                                        | Description                      |
| -------- | ------------------------------------------- | -------------------------------- |
| `entity` | [`EntityManager`](/sdk/types/EntityManager) | The entity that has been created |

### Example

```js
engine.players.events.on('onCreate', player => {
  console.log('A player has been created with id: ' + player.id);
});
```

**Available on:** <ClientTag /> <ServerTag />
