This event is triggered when an entity is streamed in.

### Parameters

| Name     | Type            | Description                           |
| -------- | --------------- | ------------------------------------- |
| `entity` | `EntityManager` | The entity that has been streamed in. |

### Example

```js
engine.players.events.on('onStreamIn', player => {
  console.log('A player has streamed in: ' + player.id);
});
```

**Available on:** <ClientTag /> <ServerTag />
