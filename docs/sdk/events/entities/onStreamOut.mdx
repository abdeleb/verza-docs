This event is triggered when an entity is streamed out.

### Parameters

| Name     | Type            | Description                            |
| -------- | --------------- | -------------------------------------- |
| `entity` | `EntityManager` | The entity that has been streamed out. |

### Example

```js
engine.players.events.on('onStreamOut', player => {
  console.log('A player has streamed out: ' + player.id);
});
```

**Available on:** <ClientTag /> <ServerTag />
