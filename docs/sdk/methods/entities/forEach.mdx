Calls a provided function once for each entity, similar to [`Array.forEach`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/forEach).

### Syntax

```js
entities.forEach(callback);
```

### Parameters

| Name       | Type                              | Description |
| ---------- | --------------------------------- | ----------- |
| `callback` | `(entity: EntityManager) => void` | Callback    |

### Example

```js
engine.players.forEach(player => {
  player.sendMessage('Hello!');
});
```
