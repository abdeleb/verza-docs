Unregisters a command.

### Syntax

```js
engine.commands.unregister(command);
```

### Parameters

| Name      | Type                            | Description               |
| --------- | ------------------------------- | ------------------------- |
| `command` | [`Command`](/sdk/types/Command) | The command to unregister |

### Example

```js
engine.commands.unregister(command);
```

**Available on:** <ClientTag /> <ServerTag />
