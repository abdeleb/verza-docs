Sets the render mode of the viewport.

### Syntax

```js
engine.world.setViewportRender(mode);
```

### Parameters

| Name   | Type                                                 | Description      |
| ------ | ---------------------------------------------------- | ---------------- |
| `mode` | [`ViewportRenderMode`](/sdk/types/setViewportRender) | The render mode. |

### Example

```js
engine.world.setViewportRender('wireframe');
```
