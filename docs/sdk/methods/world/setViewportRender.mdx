Sets the render mode of the viewport.

### Syntax

```js
engine.world.setViewportRender(mode);
```

### Parameters

| Name   | Type                                          | Description                           |
| ------ | --------------------------------------------- | ------------------------------------- |
| `mode` | [`ViewportRender`](/sdk/types/ViewportRender) | The render mode. Default is `render`. |

### Example

```js
engine.world.setViewportRender('wireframe');
```
