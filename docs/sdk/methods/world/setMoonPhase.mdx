Sets the moon phase.

### Syntax

```js
engine.world.setMoonPhase(phase);
```

### Parameters

| Name    | Type                                   | Description            |
| ------- | -------------------------------------- | ---------------------- |
| `phase` | [`MoonPhases`](./sdk/types/MoonPhases) | The moon phase to set. |

### Example

```js
engine.world.setMoonPhase('FULL_MOON');
```
