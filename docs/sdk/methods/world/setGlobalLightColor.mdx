Sets the color of the global light.

### Syntax

```js
engine.world.setGlobalLightColor(seconds);
```

### Parameters

| Name    | Type                                | Description                    |
| ------- | ----------------------------------- | ------------------------------ |
| `color` | [`ColorType`](/sdk/types/ColorType) | The color of the global light. |

### Example

```js
// set the time to 10am
engine.world.setGlobalLightColor('red');
```
