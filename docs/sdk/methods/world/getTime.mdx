Gets the current time of day in seconds.

### Syntax

```js
engine.world.getTime();
```

### Returns

- `Promise<boolean>` - A promise that resolves to the current time of day in seconds.

### Example

```js
// set the time to 10am
const time = await engine.world.getTime();

console.log('Current time', time);
```
