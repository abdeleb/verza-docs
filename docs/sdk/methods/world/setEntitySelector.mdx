The entity selector allows the player to select entities in the game world by clicking on them.

### Syntax

```js
engine.world.setEntitySelector(status);
```

### Parameters

| Name     | Type      | Description                            |
| -------- | --------- | -------------------------------------- |
| `status` | `boolean` | Enable or disable the entity selector. |

### Example

```js
engine.world.setEntitySelector(true);
```
