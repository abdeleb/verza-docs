Set the weather in the world.

### Syntax

```js
engine.world.setWeather(weather);
```

### Parameters

| Name      | Type                                     | Description         |
| --------- | ---------------------------------------- | ------------------- |
| `weather` | [`WeatherType`](./sdk/types/WeatherType) | The weather to set. |

### Example

```js
// Set the weather to cloudy
engine.world.setWeather('cloudy');
```
