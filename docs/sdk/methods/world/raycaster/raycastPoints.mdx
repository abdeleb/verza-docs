Casts rays from two positions in the world and returns hit information.

### Syntax

```js
engine.world.raycastPoint(from, to, options);
```

### Parameters

| Name      | Type                                                                         | Description                                            |
| --------- | ---------------------------------------------------------------------------- | ------------------------------------------------------ |
| `from`    | [`Vector3`](/sdk/types/Vector3) or [`Vector3Array`](/sdk/types/Vector3Array) | The starting position of the ray in world coordinates. |
| `to`      | [`Vector3`](/sdk/types/Vector3) or [`Vector3Array`](/sdk/types/Vector3Array) | The ending position of the ray in world coordinates.   |
| `options` | [`RaycastOptions`](/sdk/types/RaycastOptions)                                | Additional options for the raycast.                    |

### Returns

- [`IntersectsResult`](/sdk/types/IntersectsResult) - The result of the raycast.

### Example

```js
const intersects = engine.world.raycastPoint(
  [5, 1, 10], // from - where to start the ray
  [10, 1, 20], // to - where to end the ray
);

console.log('what did we hit?', intersects);
```
