Set the mass of the object.

### Syntax

```js
object.setMass(mass);
```

### Parameters

| Name   | Type     | Description  |
| ------ | -------- | ------------ |
| `mass` | `number` | The new mass |

### Example

```js
object.setMass(100);
```

**Available on:** <ClientTag /> <ServerTag />
