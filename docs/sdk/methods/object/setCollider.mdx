Set the collider type of the object.

### Syntax

```js
object.setCollider(type);
```

### Parameters

| Name   | Type                                          | Description           |
| ------ | --------------------------------------------- | --------------------- |
| `type` | [`EntityCollider`](/sdk/types/EntityCollider) | The new collider type |

### Example

```js
object.setCollider('hull');
```

**Available on:** <ClientTag /> <ServerTag />
