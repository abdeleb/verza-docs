Starts a transition for the object.

### Syntax

```js
object.startTransition(transition);
```

### Parameters

| Name         | Type                                              | Description                                 |
| ------------ | ------------------------------------------------- | ------------------------------------------- |
| `transition` | [`CameraTransition`](/sdk/types/CameraTransition) | The transition to be applied to the object. |

### Example

```js
object.startTransition({
  from: [0, 5, 10],
  to: [0, 5, 15],
  duration: 500,
});
```

**Available on:** <ClientTag /> <ServerTag />
