Adds torque to the object.

### Syntax

```js
object.addTorque(vector);
```

### Parameters

| Name     | Type                                      | Description                                  |
| -------- | ----------------------------------------- | -------------------------------------------- |
| `vector` | [`Vector3Array`](/sdk/types/Vector3Array) | The torque vector to be added to the object. |

### Example

```js
object.addTorque([0, 20, 0]);
```
