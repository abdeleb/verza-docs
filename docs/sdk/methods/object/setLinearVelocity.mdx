Sets the linear velocity of the object.

### Syntax

```js
object.setLinearVelocity(vector);
```

### Parameters

| Name     | Type                                      | Description                 |
| -------- | ----------------------------------------- | --------------------------- |
| `vector` | [`Vector3Array`](/sdk/types/Vector3Array) | The linear velocity vector. |

### Example

```js
// make it jump
object.setLinearVelocity([0, 20, 0]);
```

**Available on:** <ClientTag /> <ServerTag />
