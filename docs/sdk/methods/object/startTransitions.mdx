Starts all specified transitions for the object.

### Syntax

```js
object.startTransitions(transitions);
```

### Parameters

| Name          | Type                                                | Description                                  |
| ------------- | --------------------------------------------------- | -------------------------------------------- |
| `transitions` | [`ObjectTransition[]`](/sdk/types/ObjectTransition) | The transitions to be applied to the object. |

### Example

```js
object.startTransitions([
  {
    from: [0, 5, 10],
    to: [0, 5, 15],
    duration: 500,
    easing: 'easeOutQuad',
  },
  {
    to: [0, 5, 10],
    duration: 500,
    easing: 'easeInQuad',
  },
]);
```
