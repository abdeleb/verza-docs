Update data for the object.

### Syntax

```js
object.setData(data);
```

### Parameters

| Name    | Type         | Description    |
| ------- | ------------ | -------------- |
| `props` | `ObjectData` | Data to update |

### Example

```js
object.setData({
  collision: 'dynamic',
});
```

**Available on:** <ClientTag /> <ServerTag />
