Updates type-specific properties for an object.

### Syntax

```js
object.setProps(props);
```

### Parameters

| Name    | Type          | Description          |
| ------- | ------------- | -------------------- |
| `props` | `ObjectProps` | Properties to update |

### Example

```js
object.setProps({
  color: 'blue',
});
```
