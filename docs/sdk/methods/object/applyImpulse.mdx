Applies an impulse to the object.

### Syntax

```js
object.applyImpulse(vector);
```

### Parameters

| Name     | Type                                      | Description                                     |
| -------- | ----------------------------------------- | ----------------------------------------------- |
| `vector` | [`Vector3Array`](/sdk/types/Vector3Array) | The impulse vector to be applied to the object. |

### Example

```js
// make it spin
object.applyImpulse([0, 20, 0]);
```
