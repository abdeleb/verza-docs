Sets the friction of the physics object.

### Syntax

```js
object.setFriction(friction);
```

### Parameters

| Name       | Type     | Description                         |
| ---------- | -------- | ----------------------------------- |
| `friction` | `number` | The friction of the physics object. |

### Example

```js
object.setFriction(20);
```

**Available on:** <ClientTag /> <ServerTag />
