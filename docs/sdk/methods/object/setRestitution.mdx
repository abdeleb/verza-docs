Sets the restitution of the physics object.

### Syntax

```js
object.setRestitution(restitution);
```

### Parameters

| Name          | Type     | Description                           |
| ------------- | -------- | ------------------------------------- |
| `restitution` | `number` | The restitution of the physics object |

### Example

```js
object.setRestitution(20);
```

**Available on:** <ClientTag /> <ServerTag />
