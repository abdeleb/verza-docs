Set the rotation of the object in world space.

### Syntax

```js
object.setRotationFromWorldSpace(rotation);
```

### Parameters

| Name       | Type                                                                                                                                                         | Description      |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------- |
| `rotation` | [`Euler`](/sdk/types/Euler), [`EulerArray`](/sdk/types/EulerArray), [`Quaternion`](/sdk/types/Quaternion) or [`QuaternionArray`](/sdk/types/QuaternionArray) | The new rotation |

### Example

```js
object.setRotationFromWorldSpace([0, Math.PI, 0]);
```

**Available on:** <ClientTag /> <ServerTag />
