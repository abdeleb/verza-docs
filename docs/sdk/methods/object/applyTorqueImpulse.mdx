Applies an impulse torque to the object.

### Syntax

```js
object.applyTorqueImpulse(vector);
```

### Parameters

| Name     | Type                                      | Description                                            |
| -------- | ----------------------------------------- | ------------------------------------------------------ |
| `vector` | [`Vector3Array`](/sdk/types/Vector3Array) | The impulse torque vector to be applied to the object. |

### Example

```js
object.applyTorqueImpulse([0, 20, 0]);
```
