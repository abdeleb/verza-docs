Sets the angular velocity of the object.

### Syntax

```js
object.setAngularVelocity(vector);
```

### Parameters

| Name     | Type                                      | Description                  |
| -------- | ----------------------------------------- | ---------------------------- |
| `vector` | [`Vector3Array`](/sdk/types/Vector3Array) | The angular velocity vector. |

### Example

```js
// make it spin
object.setAngularVelocity([0, 20, 0]);
```

**Available on:** <ClientTag /> <ServerTag />
