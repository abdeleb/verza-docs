Restarts the server instance.

### Syntax

```js
engine.restartServer(reason);
```

### Parameters

| Name     | Type     | Description       |
| -------- | -------- | ----------------- |
| `reason` | `string` | Reason (optional) |

### Example

```js
engine.restartServer('Server restarting, will be back in a moment!');
```

**Available on:** <ServerTag />
