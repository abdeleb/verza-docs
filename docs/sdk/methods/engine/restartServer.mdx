Restarts the server instance.

### Syntax

```js
engine.restartServer(reason);
```

### Parameters

| Name     | Type     | Description                                                                                  |
| -------- | -------- | -------------------------------------------------------------------------------------------- |
| `reason` | `string` | An optional string message that will be logged by Verza for debugging or reference purposes. |

### Example

```js
engine.restartServer('Server restarting, please wait...');
```

**Available on:** <ServerTag />
