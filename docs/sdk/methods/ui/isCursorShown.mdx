Checks whether the cursor is shown.

### Syntax

```js
engine.ui.isCursorShown();
```

### Returns

- `boolean` - `true` if the cursor is shown, `false` otherwise.

### Example

```js
console.log(
  `The cursor is currently being shown: ${engine.ui.isCursorShown()}`,
);
```

**Available on:** <ClientTag />
