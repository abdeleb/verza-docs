Hides a UI component.

### Syntax

```js
engine.ui.hideComponent(componentId);
```

### Parameters

| Name          | Type                                            | Description                           |
| ------------- | ----------------------------------------------- | ------------------------------------- |
| `componentId` | [`UIComponentType`](/sdk/types/UIComponentType) | The type of the UI component to hide. |

### Example

```js
engine.ui.hideComponent('chat');
```

**Available on:** <ClientTag />
