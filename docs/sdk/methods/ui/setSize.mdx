Sets the size properties of the UI.

### Syntax

```js
engine.ui.setProps(props);
```

### Parameters

| Name    | Type                                    | Description                           |
| ------- | --------------------------------------- | ------------------------------------- |
| `props` | [`UISizeProps`](/sdk/types/UISizeProps) | The size properties to set for the UI |

### Example

```js
engine.ui.setProps({
  width: '200px',
  height: '500px',

  right: '20vw',
  bottom: '5vh',
});
```
