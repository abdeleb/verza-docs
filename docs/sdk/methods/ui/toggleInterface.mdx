Toggles whether an interface is added or not

### Syntax

```js
engine.ui.toggleInterface(interface);
```

### Parameters

| Name        | Type     | Description        |
| ----------- | -------- | ------------------ |
| `interface` | `string` | The interface name |

### Example

```js
engine.ui.toggleInterface('my-interface');
```

**Available on:** <ClientTag />
