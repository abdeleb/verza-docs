Goes to a server.

### Syntax

```js
engine.ui.goToServer(serverId);
```

### Parameters

| Name       | Type     | Description                    |
| ---------- | -------- | ------------------------------ |
| `serverId` | `string` | The ID of the server to go to. |

### Example

```js
engine.ui.goToServer('server_1vuuMRTCDX9frQORQSxtVa');
```
