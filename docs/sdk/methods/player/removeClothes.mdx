Removes clothes from the player.

### Syntax

```js
player.removeClothes(ids);
```

### Parameters

| Name  | Type       | Description                        |
| ----- | ---------- | ---------------------------------- |
| `ids` | `string[]` | An array of clothes IDs to remove. |

### Example

```js
player.addClothes(['beach-tshirt']);
```

**Available on:** <ClientTag /> <ServerTag />
