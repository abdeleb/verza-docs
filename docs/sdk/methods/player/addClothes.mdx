Adds clothes to the player.

### Syntax

```js
player.addClothes(clothes);
```

### Parameters

| Name      | Type                                                | Description                       |
| --------- | --------------------------------------------------- | --------------------------------- |
| `clothes` | [`PlayerClotheItem[]`](/types/sdk/PlayerClotheItem) | The clothes to add to the player. |
| `replace` | `boolean` (optional)                                | Whether to replace all clothes.   |

### Example

```js
player.addClothes([
  {
    id: 'beach-tshirt',
  },
]);
```
