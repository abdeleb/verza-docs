Sets the player's facing angle.

### Syntax

```js
player.getFacingAngle();
```

### Returns

- `angle` - The angle the player is facing.

### Example

```js
console.log('Player is facing ' + player.getFacingAngle() + ' degrees');
```

**Available on:** <ClientTag /> <ServerTag />
