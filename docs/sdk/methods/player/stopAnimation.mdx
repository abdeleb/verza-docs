Stops an animation on the player.

### Syntax

```js
player.stopAnimation(id);
```

### Parameters

| Name     | Type     | Description                      |
| -------- | -------- | -------------------------------- |
| `animId` | `string` | The ID of the animation to stop. |

### Example

```js
player.stopAnimation('dance');
```

**Available on:** <ClientTag /> <ServerTag />
