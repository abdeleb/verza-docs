Sets the player's movement controls.

### Syntax

```js
player.setMovements(status);
```

### Parameters

| Name     | Type      | Description                                                           |
| -------- | --------- | --------------------------------------------------------------------- |
| `status` | `boolean` | Whether the player's movement controls should be enabled or disabled. |

### Example

```js
// block all player movements
player.setMovements(false);
```

**Available on:** <ClientTag /> <ServerTag />
