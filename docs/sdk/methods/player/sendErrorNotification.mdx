Sends an error notification to the player.

### Syntax

```js
player.sendErrorNotification(message);
```

### Parameters

| Name      | Type     | Description                            |
| --------- | -------- | -------------------------------------- |
| `message` | `string` | The notification to send to the player |

### Example

```js
player.sendErrorNotification('You were removed from the Team #1');
```

**Available on:** <ClientTag /> <ServerTag />
