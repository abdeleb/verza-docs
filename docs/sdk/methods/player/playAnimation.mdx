Plays an animation on the player.

### Syntax

```js
player.playAnimation(id, options);
```

### Parameters

| Name      | Type                                                | Description                          |
| --------- | --------------------------------------------------- | ------------------------------------ |
| `animId`  | `string`                                            | The ID of the animation to play.     |
| `options` | [`AnimationOptions[]`](/types/sdk/AnimationOptions) | The animation to play on the player. |

### Example

```js
player.playAnimation('dance', {
  loop: 'repeat';
});
```

**Available on:** <ClientTag /> <ServerTag />
