Sets the camera fov for the player.

:::note
This method is only available in `global` mode.
:::

### Syntax

```js
player.camera.setFov(transitions);
```

### Parameters

| Name  | Type               | Description                                 |
| ----- | ------------------ | ------------------------------------------- |
| `fov` | `number` or `null` | The fov to set, `null` to reset to default. |

### Example

```js
player.camera.setFov(120);
```

**Available on:** <ClientTag /> <ServerTag />
