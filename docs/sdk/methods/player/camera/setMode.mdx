Sets the camera mode for the player.

### Syntax

```js
player.camera.setMode(mode);
```

### Parameters

| Name   | Type                                  | Description             |
| ------ | ------------------------------------- | ----------------------- |
| `mode` | [`CameraMode`](/sdk/types/CameraMode) | The camera mode to set. |

### Example

```js
player.camera.setMode('global');
```

**Available on:** <ClientTag /> <ServerTag />
