Starts a transition for the camera.

:::note
This method is only available in `global` mode.
:::

### Syntax

```js
player.camera.startTransition(transition);
```

### Parameters

| Name         | Type                                              | Description              |
| ------------ | ------------------------------------------------- | ------------------------ |
| `transition` | [`CameraTransition`](/sdk/types/CameraTransition) | The transition to start. |

### Example

```js
player.camera.startTransition({
  from: [0, 5, 10],
  to: [0, 5, 15],
  duration: 500,
});
```

**Available on:** <ClientTag /> <ServerTag />
