# Methods

Methods in Verza's SDK provide a powerful way to interact with various world elements, control the flow of your scripts, and build complex interactions. They are used to perform actions, retrieve information, and modify the game state according to your requirements.

In this section, you'll find a comprehensive list of methods available in the Verza SDK, along with detailed descriptions and usage examples. Discover how to effectively utilize these methods to create engaging, interactive, and immersive experiences for your players within your scripts.

## Engine

### Methods

| Method                                                   | Description                                                    |
| -------------------------------------------------------- | -------------------------------------------------------------- |
| [engine.connectServer](./engine/connectServer)           | Connects a websocket server script to a Verza server instance. |
| [engine.connectClient](./engine/connectClient)           | Connects a client script to a Verza server instance.           |
| [engine.restartServer](./engine/restartServer)           | Restarts the server instance.                                  |
| [engine.setForwardMessages](./engine/setForwardMessages) | Enables or disables message forwarding to scripts.             |
| [engine.destroy](./engine/destroy)                       | Cleans up and releases engine resources.                       |

| [chat.escapeText](./chat/escapeText)             | Escapes special characters in a message so that they are displayed correctly in the chat. |

## Entities

### Methods

| Method                                 | Description                                                                                                                                                                                                                                                        |
| -------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| [entities.is](./entities/is)           | Checks whether an entity with the specified ID exists.                                                                                                                                                                                                             |
| [entities.get](./entities/get)         | Retrieves the entity with the specified ID. Returns `null` if the entity does not exist.                                                                                                                                                                           |
| [entities.destroy](./entities/destroy) | Destroys the entity with the specified ID.                                                                                                                                                                                                                         |
| [entities.forEach](./entities/forEach) | Calls a provided function once for each entity, similar to [`Array.forEach`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/forEach).                                                                                      |
| [entities.map](./entities/map)         | Creates a new array with the results of calling a provided function on every entity, similar to [`Array.map`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/map).                                                         |
| [entities.some](./entities/some)       | Tests whether at least one entity passes the provided test, similar to [`Array.some`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/some).                                                                                |
| [entities.filter](./entities/filter)   | Creates a new array with all entities that pass the provided test, similar to [`Array.filter`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/filter).                                                                     |
| [entities.find](./entities/find)       | Returns the first entity that satisfies the provided testing function or `null`, similar to [`Array.find`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/find). If no entity matches the test, the method returns `null`. |

### Properties

| Name                   | Type                                  | Description                                                                                                |
| ---------------------- | ------------------------------------- | ---------------------------------------------------------------------------------------------------------- |
| `entities.type`        | [`EntityType`](/sdk/types/EntityType) | The type of entity managed by the entity manager.                                                          |
| `entities.entities`    | `EntityManager[]`                     | An array containing all the entity managers for the specified entity type.                                 |
| `entities.entitiesMap` | `Map<ID, EntityManager>`              | A map of entity managers indexed by their respective entity IDs.                                           |
| `entities.streamed`    | `Map<ID, EntityManager>`              | A map of entity managers for entities that are currently being streamed to the player.                     |
| `entities.events`      | `EventsManager`                       | An instance of the `EventsManager` class used to register and manage events for the specified entity type. |
