# Functions

Functions in Verza's SDK provide a powerful way to interact with various world elements, control the flow of your scripts, and build complex interactions. They are used to perform actions, retrieve information, and modify the game state according to your requirements.

In this section, you'll find a comprehensive list of functions available in the Verza SDK, along with detailed descriptions and usage examples. Discover how to effectively utilize these functions to create engaging, interactive, and immersive experiences for your players within your scripts.
