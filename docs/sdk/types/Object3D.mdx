This type represents an object in a 3D scene using the Object3D class from the Three.js library. You can learn more about Object3D in the official Three.js documentation: https://threejs.org/docs/#api/en/core/Object3D

```ts
type Object3D = {
  position: Vector3;

  rotation: Euler;

  quaternion: Quaternion;

  scale: Vector3;
};
```
