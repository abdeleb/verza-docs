`Capsule` is a primitive object that creates a capsule.

### Syntax

```jsx
<Capsule />
```

### Parameters

All objects share the same base parameters from [`ComponentObjectProps`](/sdk/types/ComponentObjectProps).

| Name             | Type                                                | Description                                       |
| ---------------- | --------------------------------------------------- | ------------------------------------------------- |
| `radius`         | `number`                                            | The radius of the capsule.                        |
| `length`         | `number`                                            | The length of the capsule.                        |
| `capSegments`    | `number`                                            | The number of segments of the capsule's caps.     |
| `radialSegments` | `number`                                            | The number of segments of the capsule's cylinder. |
| `color`          | [`ColorType`](/sdk/types/ColorType)                 | The color of the box.                             |
| `material`       | [`ObjectStandardMix`](/sdk/types/ObjectStandardMix) | The material of the box.                          |

### Example

```jsx
<Capsule radius={0.5} length={1} />
```
