`Circle` is a primitive object that creates a circle.

### Syntax

```jsx
<Circle />
```

### Parameters

All objects share the same base parameters from [`ComponentObjectProps`](/sdk/types/ComponentObjectProps).

| Name          | Type                                                | Description                           |
| ------------- | --------------------------------------------------- | ------------------------------------- |
| `radius`      | `number`                                            | The radius of the circle.             |
| `segments`    | `number`                                            | The number of segments of the circle. |
| `thetaStart`  | `number`                                            | The starting angle of the circle.     |
| `thetaLength` | `number`                                            | The length of the circle.             |
| `bloom`       | `boolean`                                           | Whether use bloom effect.             |
| `color`       | [`ColorType`](/sdk/types/ColorType)                 | The base color.                       |
| `material`    | [`ObjectMaterialMix`](/sdk/types/ObjectMaterialMix) | The material.                         |

### Example

```jsx
<Circle radiusTop={0.5} radiusBottom={0.5} height={1} />
```
