`Torus` is a primitive object that creates a torus.

### Syntax

```jsx
<Torus />
```

### Parameters

All objects share the same base parameters from [`ComponentObjectProps`](/sdk/types/ComponentObjectProps).

| Name              | Type                                                          | Description                                         |
| ----------------- | ------------------------------------------------------------- | --------------------------------------------------- |
| `radius`          | `number`                                                      | The radius of the torus.                            |
| `tube`            | `number`                                                      | The tube radius of the torus.                       |
| `radialSegments`  | `number`                                                      | The number of segments along the radial direction.  |
| `tubularSegments` | `number`                                                      | The number of segments along the tubular direction. |
| `arc`             | `number`                                                      | The central angle of the torus.                     |
| `color`           | [`ColorType`](/sdk/types/ColorType)                           | The color of the box.                               |
| `material`        | [`ObjectStandardMaterial`](/sdk/types/ObjectStandardMaterial) | The material of the box.                            |

### Example

```jsx
<Torus
  radius={1}
  tube={0.4}
  radialSegments={8}
  tubularSegments={6}
  arc={Math.PI * 2}
  color="#00ff00"
/>
```
