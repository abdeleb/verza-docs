`Gltf` is a component that allows you to load a GLTF model into the world.

### Syntax

```jsx
<Gltf url={url} />
```

### Parameters

All objects share the same base parameters from [`CreateObjectProps`](/sdk/types/CreateObjectProps).

| Name  | Type     | Description                                                                   |
| ----- | -------- | ----------------------------------------------------------------------------- |
| `url` | `string` | The URL of the GLTF model to load, it could be an `asset_id` or a remote URL. |

### Example

```jsx
<Gltf url="glb url" />
```

**Available on:** <ClientTag /> <ServerTag />
