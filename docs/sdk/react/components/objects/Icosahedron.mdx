`Icosahedron` is a primitive object that creates a icosahedron.

### Syntax

```jsx
<Icosahedron />
```

### Parameters

All objects share the same base parameters from [`ComponentObjectProps`](/sdk/types/ComponentObjectProps).

| Name       | Type                                                | Description                    |
| ---------- | --------------------------------------------------- | ------------------------------ |
| `radius`   | `number`                                            | The radius of the icosahedron. |
| `detail`   | `number`                                            | The detail of the icosahedron. |
| `color`    | [`ColorType`](/sdk/types/ColorType)                 | The color of the box.          |
| `material` | [`ObjectMaterialMix`](/sdk/types/ObjectMaterialMix) | The material of the box.       |

### Example

```jsx
<Icosahedron radius={1} detail={0} color="red" />
```
