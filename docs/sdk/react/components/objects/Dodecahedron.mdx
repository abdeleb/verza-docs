`Dodecahedron` is a primitive object that creates a dodecahedron.

### Syntax

```jsx
<Dodecahedron />
```

### Parameters

All objects share the same base parameters from [`ComponentObjectProps`](/sdk/types/ComponentObjectProps).

| Name       | Type                                                | Description                     |
| ---------- | --------------------------------------------------- | ------------------------------- |
| `radius`   | `number`                                            | The radius of the dodecahedron. |
| `detail`   | `number`                                            | The detail of the dodecahedron. |
| `color`    | [`ColorType`](/sdk/types/ColorType)                 | The color of the box.           |
| `material` | [`ObjectStandardMix`](/sdk/types/ObjectStandardMix) | The material of the box.        |

### Example

```jsx
<Dodecahedron radius={1} detail={0} color="red" />
```
