`Line` is a primitive object that creates a line.

### Syntax

```jsx
<Line />
```

### Parameters

All objects share the same base parameters from [`ComponentObjectProps`](/sdk/types/ComponentObjectProps).

| Name     | Type                                        | Description             |
| -------- | ------------------------------------------- | ----------------------- |
| `points` | [`Vector3Array[]`](/sdk/types/Vector3Array) | The points of the line. |
| `color`  | [`ColorType`](/sdk/types/ColorType)         | The color of the line.  |

### Example

```jsx
<Line
  points={[
    [0, 2, 10],
    [1, 1, 5],
    [3, 4, 5],
  ]}
  position={[0, 2, 0]}
/>
```
