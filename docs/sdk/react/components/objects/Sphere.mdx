`Sphere` is a primitive object that creates a sphere.

### Syntax

```jsx
<Sphere />
```

### Parameters

All objects share the same base parameters from [`ComponentObjectProps`](/sdk/types/ComponentObjectProps).

| Name             | Type                                                | Description                                            |
| ---------------- | --------------------------------------------------- | ------------------------------------------------------ |
| `radius`         | `number`                                            | The radius of the sphere.                              |
| `widthSegments`  | `number`                                            | The number of segments along the width of the sphere.  |
| `heightSegments` | `number`                                            | The number of segments along the height of the sphere. |
| `bloom`          | `boolean`                                           | Whether use bloom effect.                              |
| `color`          | [`ColorType`](/sdk/types/ColorType)                 | The base color.                                        |
| `material`       | [`ObjectMaterialMix`](/sdk/types/ObjectMaterialMix) | The material.                                          |

### Example

```jsx
<Sphere radius={1} position={[0, 2, 0]} />
```
