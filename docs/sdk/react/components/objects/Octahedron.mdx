`Octahedron` is a primitive object that creates a octahedron.

### Syntax

```jsx
<Octahedron />
```

### Parameters

All objects share the same base parameters from [`ComponentObjectProps`](/sdk/types/ComponentObjectProps).

| Name       | Type                                                | Description                   |
| ---------- | --------------------------------------------------- | ----------------------------- |
| `radius`   | `number`                                            | The radius of the octahedron. |
| `detail`   | `number`                                            | The detail of the octahedron. |
| `bloom`    | `boolean`                                           | Whether use bloom effect.     |
| `color`    | [`ColorType`](/sdk/types/ColorType)                 | The base color.               |
| `material` | [`ObjectMaterialMix`](/sdk/types/ObjectMaterialMix) | The material.                 |

### Example

```jsx
<Octahedron radius={1} detail={0} color="red" />
```
