# React

The Verza SDK React library is an extension of the core Verza SDK, designed to streamline the development process for frontend developers using the popular React framework. With a set of custom hooks and components, this library allows you to create and manage game objects, handle events, and interact with the Verza API in a more declarative and React-friendly way.
