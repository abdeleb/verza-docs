Uses an object by their ID.

### Syntax

```js
useObject(id);
```

| Name | Type     | Description                  |
| ---- | -------- | ---------------------------- |
| `id` | `string` | The ID of the object to use. |

### Returns

- [`ObjectManager](/sdk/types/ObjectManager) - The object manager.

### Example

```js
const object = useObject('my-object-id');
```

**Available on:** <ClientTag /> <ServerTag />
