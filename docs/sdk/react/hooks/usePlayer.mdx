Uses a player by their ID.

### Syntax

```js
usePlayer(id);
```

| Name | Type     | Description                  |
| ---- | -------- | ---------------------------- |
| `id` | `number` | The ID of the player to use. |

### Example

```js
const player = usePlayer(1);
```

**Available on:** <ClientTag /> <ServerTag />
