Listen to an event.

### Syntax

```js
useEvent();
```

### Parameters

| Name        | Type       | Description                                       |
| ----------- | ---------- | ------------------------------------------------- |
| `eventName` | `string`   | The name of the event to listen to.               |
| `callback`  | `function` | The callback to call when the event is triggered. |

### Example

```js
useEvent('onKeyDown', event => {
  console.log('key down!', event);
});
```

**Available on:** <ClientTag /> <ServerTag />
