Adds a command to the server.

### Syntax

```js
useCommand(command);
```

### Parameters

| Name      | Type                            | Description         |
| --------- | ------------------------------- | ------------------- |
| `command` | [`Command`](/sdk/types/Command) | The command to add. |

### Example

```js
useCommand(
  new Command('test', player => {
    player.sendMessage('This is a test!');
  }),
);
```

**Available on:** <ClientTag /> <ServerTag />
