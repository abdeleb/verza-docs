Adds a sound to the engine.

### Syntax

```js
useAddSound(item);
```

### Parameters

| Name   | Type        | Description                  |
| ------ | ----------- | ---------------------------- |
| `item` | `SoundItem` | The ID of the player to use. |

### Example

```js
useAddSound({
  soundName: 'fire',
  url: 'https://www.example.com/fire.mp3',
});
```

**Available on:** <ClientTag />
