Returns the local player streamed in.

### Syntax

```js
useStreamedLocalPlayer();
```

### Returns

- [`PlayerManager](/sdk/types/PlayerManager) - The player manager.

### Example

```js
const localPlayer = useStreamedLocalPlayer();
```

**Available on:** <ClientTag />
