Returns the local player streamed in.

### Syntax

```js
useStreamedLocalPlayer();
```

### Returns

- [`PlayerManager`](/sdk/types/PlayerManager) or `null` - The player manager of the local player streamed in, or `null` if the local player is not streamed in.

### Example

```js
const localPlayer = useStreamedLocalPlayer();
```

**Available on:** <ClientTag />
