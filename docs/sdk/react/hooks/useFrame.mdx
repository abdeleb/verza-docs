Calls a function every frame.

### Syntax

```js
useFrame(callback);
```

### Parameters

| Name       | Type                      | Description                                                                                          |
| ---------- | ------------------------- | ---------------------------------------------------------------------------------------------------- |
| `callback` | `(delta: number) => void` | The function to call every frame. The `delta` parameter is the time in seconds since the last frame. |

### Example

```js
useFrame(delta => {
  // delta...
});
```

**Available on:** <ClientTag />
