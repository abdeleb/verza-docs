Calls a function every frame.

### Syntax

```js
useFrame(callback);
```

### Parameters

| Name       | Type                      | Description                       |
| ---------- | ------------------------- | --------------------------------- |
| `callback` | `(delta: number) => void` | The function to call every frame. |

### Example

```js
useFrame(() => {
  // ...
});
```

**Available on:** <ClientTag />
