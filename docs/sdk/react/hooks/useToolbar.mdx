Adds a toolbar to the UI.

### Syntax

```js
useToolbar();
```

### Parameters

| Name      | Type                                          | Description         |
| --------- | --------------------------------------------- | ------------------- |
| `toolbar` | [`ToolbarElement`](/sdk/types/ToolbarElement) | The toolbar to add. |

### Example

```js
useToolbar({
  id: 'my-toolbar',
  position: 'bottom',
  items: [
    {
      name: 'Open House Menu',
      key: 'Y',
    },
  ],
});
```
