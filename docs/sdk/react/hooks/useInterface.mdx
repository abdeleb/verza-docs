Detects if the specified interface is present.

### Syntax

```js
useInterface(interface);
```

### Parameters

| Name        | Type     | Description                 |
| ----------- | -------- | --------------------------- |
| `interface` | `string` | The interface to check for. |

### Example

```js
const isInterface = useInterface(interface);
```

**Available on:** <ClientTag />
