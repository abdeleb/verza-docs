Adds a toolbar item to the main toolbar.

### Syntax

```js
useMainToolbarItem(item);
```

### Parameters

| Name   | Type                                            | Description              |
| ------ | ----------------------------------------------- | ------------------------ |
| `item` | [`MainToolbarItem`](/sdk/types/MainToolbarItem) | The toolbar item to add. |

### Example

```js
useMainToolbarItem({
  name: 'Fly Mode',
  key: 'U',
});
```

**Available on:** <ClientTag />
