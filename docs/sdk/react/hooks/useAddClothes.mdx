Adds clothes to the clothes system.

:::caution Heads Up!
This hook do not add the clothes to the player, it only adds it to the clothes system. To add the clothes to the player, use [`player.addClothes`](/sdk/managers/player/addClothes).
:::

### Syntax

```js
useAddClothes(clothes);
```

### Parameters

| Name      | Type                                    | Description                       |
| --------- | --------------------------------------- | --------------------------------- |
| `clothes` | [`ClotheItem[]`](/sdk/types/ClotheItem) | The clothes to add to the engine. |

### Example

```js
useAddClothe([
  {
    id: 'beach-tshirt',
    url: 'https://example.com/beach-tshirt.glb',
    name: 'Beach T-Shirt',
    masks: ['top'],
  },
]);
```
