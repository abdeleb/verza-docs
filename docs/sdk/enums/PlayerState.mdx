| Name    | Value | Description                                            |
| ------- | ----- | ------------------------------------------------------ |
| idle    | 0     | Player is not moving and is in a stationary state.     |
| dead    | 1     | Player has died or is in a non-recoverable state.      |
| walking | 2     | Player is walking.                                     |
| running | 3     | Player is running.                                     |
| jumping | 4     | Player is jumping or in mid-air.                       |
| falling | 5     | Player is falling from a height or descending rapidly. |
